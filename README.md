# MFC Sandbox

Collection of MFC (Microsoft Foundation Classes) C++ code samples and demos.

## Overview

A large archive of Visual C++ / MFC sample projects, demos, and experiments. Covers a wide range of topics including UI controls, threading, COM/ActiveX, Crystal Reports, PDF generation, XML parsing, and more.

## Highlights

- Custom UI controls (gradient statics, colored buttons, list views)
- Multithreaded animation and worker thread examples
- COM interop and ActiveX tutorials
- Crystal Reports integration
- PDF creation and manipulation
- XML reading and XSLT processing
- Memory leak testing utilities

## Usage

Open individual project folders in Visual Studio. Most projects target older MFC/Win32 toolchains (Visual Studio 6 through 2008 era).
